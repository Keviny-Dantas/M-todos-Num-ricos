# Métodos Numéricos e Processamento de Imagens 🧮💻

Bem-vindo ao meu repositório de implementações matemáticas e computacionais! 

Este projeto concentra uma série de algoritmos e resoluções de problemas desenvolvidos durante o curso de Engenharia de Computação (IFPB). O foco principal é a construção de soluções para álgebra linear, cálculo numérico e suas aplicações práticas, como o processamento e restauração de imagens.

## 🎯 Objetivo
Demonstrar a aplicação de lógica de programação e matemática computacional para resolver problemas complexos, manipular estruturas de dados (matrizes e vetores) e otimizar cálculos, habilidades essenciais para Análise de Dados e Data Science.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Linguagem:** Python
* **Ambiente de Desenvolvimento:** Jupyter Notebook / Google Colab
* **Bibliotecas Principais:** Pandas, NumPy, Matplotlib *(adicione outras se tiver usado, como OpenCV)*

## 📂 Estrutura dos Projetos

Os notebooks estão divididos em aplicações lógicas e resoluções de problemas reais:

### 1. Fundamentos e Álgebra Linear
* `Atividade_2_2_Revisão_de_Vetores_.ipynb`: Manipulação e operações com vetores.
* `Atividade_2_3_Revisão_de_Matrizes.ipynb`: Estruturação e cálculos com matrizes bidimensionais.

### 2. Resolução de Sistemas Lineares
* `Atividade_2_4_Sistemas_Triviais.ipynb`: Soluções básicas de sistemas.
* `Atividade_2_5` e `2_6_Eliminação_Gaussiana.ipynb`: Implementação do algoritmo de Eliminação de Gauss para resolução de equações.
* `ATIVIDADE_3_1_Decomposição_LU.ipynb`: Fatoração de matrizes para otimização de sistemas complexos.

### 3. Aplicações Práticas (Imagens e Problemas Reais)
* `ATIVIDADE_3_2_Restauração_de_Imagens_Decomposição_LU.ipynb`: **Destaque do repositório.** Aplicação de métodos numéricos (Decomposição LU e Interpolação) para tratar, recuperar e processar dados de imagens.
* `ATIVIDADE_2_7_Interpolação_aplicação_sistemas.ipynb`: Uso de sistemas lineares para prever e interpolar dados.
* `Atividade_3_3_Solução_Problemas_Reais.ipynb`: Modelagem matemática aplicada a cenários do mundo real.

### 4. Busca de Raízes
* `Atividade_4_1_Método_da_Bisseção.ipynb`: Implementação de algoritmos de busca e aproximação de raízes em funções.
